# English:

# BeamMP Outbreak/Infection

This is a recreation of the Infection game mode from Dirt 3 for BeamMP

## BeamMP Installation

1. Download the newest version from the release page
2. Open the zip file and place the contents of the Client folder in a .zip folder named OutBreak and place it in the “Resources/Client” folder on the server, and place the Server files in the “Resources/Server” folder.
3. Then to start infection you can type /outbreak start or /infection start into the in game chat and hit enter
4. A full list of commands can be found with /outbreak help or /infection help

## Server Chat Commands

1. ```/infection help``` Displays List of available commands
2. ```/infection start``` Starts infection game
3. ```/infection stop``` Stops infection game
4. ```/infection reset``` Resets randomizer weights so everyone has an equal chance of getting infected again
5. ```/infection game length set``` Sets the length of the round in minutes
6. ```/infection greenFadeDist set``` Adjusts how close in meters an infected car needs to be for the screen to start going green
7. ```/infection filterIntensity set ``` Sets how intense the vignetting effect is
8. ```/infection ColorPulse toggle``` Enabling this makes the infected cars pulse between green and the original color of the car
9. ```/infection infector tint toggle``` This toggles on or off the vignetting effect on infected players
10. ```/infection ResetAtSpeedAllowed toggle``` This toggles whether players can reset at speed
11. ```/infection MaxResetSpeed set``` Sets the highest speed where resets are allowed

# Polski:

# BeamMP Outbreak/Infection

Jest to odtworzenie trybu gry Infection z gry Dirt 3 dla BeamMP.

## Instalacja BeamMP

1. Pobierz najnowszą wersję ze strony wydania.
2. Otwórz plik zip i umieść zawartość folderu Client w folderze .zip o nazwie OutBreak, a następnie umieść go w folderze „Resources/Client” na serwerze. Pliki serwera umieść w folderze „Resources/Server”.
3. Następnie, aby rozpocząć infekcję, wpisz /outbreak start lub /infection start na czacie w grze i naciśnij Enter.
4. Pełną listę poleceń można znaleźć, wpisując /outbreak help lub /infection help.

## Polecenia czatu serwera

1. ```/infection help``` Wyświetla listę dostępnych poleceń
2. ```/infection start``` Rozpoczyna grę infekcyjną
3. ```/infection stop``` Zatrzymuje grę infekcyjną
4. ```/infection reset``` Resetuje wagi losowania, aby wszyscy mieli równe szanse na ponowne zarażenie
5. ```/infection game length set``` Ustawia długość rundy w minutach
6. ```/infection greenFadeDist set``` Dostosowuje odległość w metrach, w jakiej musi znajdować się zainfekowany samochód, aby ekran zaczął zmieniać kolor na zielony
7. ```/infection filterIntensity set ``` Ustawia intensywność efektu winietowania
8. ```/infection ColorPulse toggle``` Włączenie tej opcji powoduje, że zainfekowane samochody pulsują między kolorem zielonym a oryginalnym kolorem samochodu
9. ```/infection infector tint toggle``` Włącza lub wyłącza efekt winietowania u zainfekowanych graczy
10. ```/infection ResetAtSpeedAllowed toggle``` Włącza lub wyłącza możliwość resetowania przez graczy przy określonej prędkości
11. ```/infection MaxResetSpeed set``` Ustawia maksymalną prędkość, przy której dozwolone jest resetowanie
